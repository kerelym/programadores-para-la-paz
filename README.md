# Programadores para la Paz 🕊️

Este repositorio centraliza las actividades y proyectos desarrollados durante el programa, enfocándose en ciudadanía digital, pensamiento algorítmico y herramientas de desarrollo profesional.

## 📂 Estructura del Repositorio

El contenido está organizado por carpetas semanales para facilitar la navegación:

*   **[semana1](./semana1):** Ciudadanía digital y fundamentos de algoritmos.
*   **[semana2](./semana2):** Dominio de la terminal, gestión de archivos y Git.
*   *Próximamente: Actividades de las siguientes semanas.*

---

## 🎯 Objetivos Principales

*   **Responsabilidad Digital:** Fomentar el uso ético y crítico de la información.
*   **Pensamiento Lógico:** Desarrollar algoritmos y soluciones técnicas eficientes.
*   **Control de Versiones:** Gestionar el ciclo de vida del software mediante Git y GitHub.
*   **Organización Profesional:** Mantener una estructura de archivos limpia y documentada.

---

## ⚙️ Tecnologías y Herramientas

*   **Terminal de comandos** (Navegación y gestión de archivos).
*   **Git** (Control de versiones local).
*   **GitHub** (Alojamiento y portafolio remoto).

---

## 🚀 Cómo explorar el proyecto

1.  Clona el repositorio en tu máquina local:
    ```bash
    git clone https://github.com/kerelym/programadores-para-la-paz.git
    ```
2.  Navega a la carpeta de la semana que te interese:
    ```bash
    cd semana2
    ```
