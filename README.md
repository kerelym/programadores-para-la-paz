# Programadores para la Paz 🕊️

Repositorio correspondiente a la **Actividad Semana 1**, enfocado en el desarrollo de competencias en ciudadanía digital, pensamiento algorítmico y uso de herramientas como Git y GitHub.

---

## 📌 Contenido del repositorio

Dentro de la carpeta `semana1` se encuentran los siguientes archivos:

* `preguntas-semana1.txt` → Respuestas a preguntas de selección múltiple
* `reflexion-semana1.txt` → Reflexión sobre ciudadanía digital y herramientas tecnológicas
* `algoritmo-semana1.txt` → Algoritmo para compartir información confiable en internet

---

## 🎯 Objetivo de la actividad

Comprender la relación entre:

* Uso responsable de la información
* Ciudadanía digital
* Pensamiento algorítmico
* Organización del trabajo con herramientas como Git

---

## ⚙️ Tecnologías utilizadas

* Git
* GitHub

---

## 🚀 Cómo ver el proyecto

1. Clonar el repositorio:

   ```bash
   git clone https://github.com/kerelym/programadores-para-la-paz.git
   ```

2. Abrir la carpeta `semana1` para ver los archivos de la actividad.
